-------Introduction------------------------
This is a CRUD APPLICATION, and used to make journals on daily basis. This is my summer training project given by Mr. Shushant Pathhak sir.

-------Used things-------------------------
I use HTML5, CSS3 for frontend. And i use Python programing laguage for my backend.
This project is build with the help of the Python Flask framework.
I used WAMP server for storing the database of my project.
In which i make 9-models and interconnect them in every HTML pages.

-------How to run-------------------------
1- Install Pycharm IDE.
2- Install WAMP server version 2i.
3- And most importantly Python 3.6 64-bit system.
4- Open folder in pycharm.
5- Go to setting--> interpreter--> click on "+" icon.
6- Search four libraries and install them one by one are as follows :-
	a) Flask
	b) Flask-MySQL
	c) PyMySQL
	d) SQLAlchemy
7- You have to import database files steps are as follows :-
	a) Launch WAMP server.
	b) Go to local host on the browser.
	c) Create Database name - "digitaldiary".
	d) choose import option.
	e) browse the "digitaldiary.sql" file.
	f) click save and here you go all database is import successfully.
8- Run the "main.py" file.
9- Click or copy the address of the localhost.
10- Paste it on your browser and see the results.
